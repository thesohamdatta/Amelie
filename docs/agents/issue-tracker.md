# Issue Tracker: GitHub

Issues for Project Amélie are tracked using [GitHub Issues](https://github.com/thesohamdatta/Amelie/issues).

## Skills Integration

- `to-issues`: Creates new issues via `gh issue create`.
- `triage`: Manages issue states using the label vocabulary.
- `to-prd`: Publishes PRDs to the issue tracker.

## Workflow

1.  **Creation:** Use `to-issues` to break down plans into Tracer Bullet vertical slices.
2.  **Triage:** Use the `triage` skill to evaluate incoming bugs or feature requests.
3.  **Implementation:** AFK agents pick up issues labeled `ready-for-agent`.
