# Triage Labels

Project AmĂŠlie uses a state-machine driven triage process with the following labels:

| Role | Label | Description |
| :--- | :--- | :--- |
| `needs-triage` | `needs-triage` | Newly created issues requiring maintainer evaluation. |
| `needs-info` | `needs-info` | Waiting for additional information from the reporter. |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for an autonomous agent to implement. |
| `ready-for-human` | `ready-for-human` | Requires human intervention or implementation. |
| `wontfix` | `wontfix` | Will not be actioned. |
