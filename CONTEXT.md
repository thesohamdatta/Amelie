# Project Amélie Context

## Domain Language

- **Sentient Object:** The core identity of Amélie—not an app, but a presence.
- **Soul Orb:** The central visual manifestation that reacts to audio and state.
- **Live Loop:** The real-time voice interaction pipeline (STT -> LLM -> TTS).
- **Hinglish:** The primary conversational language, a mix of Hindi and English.
- **Atmospheric Canvas:** The dark, minimalist visual environment Amélie lives in.

## Core Concepts

1. **Digital Intimacy:** Prioritizing low-latency, warm, and personal vocal interaction over transactional UI.
2. **Organic Reactivity:** Visuals (the Orb) should feel liquid and biological, not mechanical.
3. **Decoupled Brain:** The backend orchestrates separate AI services (Groq, Sarvam) into a unified experience.
