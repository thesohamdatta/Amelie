# AGENTS.md

## Project Overview

Amélie is a voice-native sentient object designed for digital intimacy. This repository is split into a **FastAPI backend** (The Brain) and a **Next.js frontend** (The Face).

## Agent skills

### Issue tracker

Issues for Project Amélie are tracked using [GitHub Issues](https://github.com/thesohamdatta/Amelie/issues). See `docs/agents/issue-tracker.md`.

### Triage labels

Project Amélie uses a state-machine driven triage process with standard labels (`needs-triage`, `ready-for-agent`, etc.). See `docs/agents/triage-labels.md`.

### Domain docs

Project Amélie follows a **Single-context** layout for domain and architectural documentation. See `docs/agents/domain.md`.
